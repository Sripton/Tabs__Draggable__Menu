CSS
white-space: nowrap; // subjects
Объединяет последовательности пробелов в один пробел, как значение normal, но не переносит строки (оборачивание текста) внутри текста.

user-select: none;
Текст элемента и вложенных в него элементов не выбирается. Обратите внимание, что объект Selection может содержать эти элементы.


this.getBoundingClientRect();
Метод Element.getBoundingClientRect() возвращает размер элемента и его позицию относительно viewport (часть страницы, показанная на экране, и которую мы видим).


JS
1. Добавление обработчика события mousemove
tabNavList.addEventListener("mousemove", (drag) => {
  if (!activeDragg) return;
  tabNavList.scrollLeft -= drag.movementX;
});

* tabNavList.addEventListener("mousemove", (drag) => { ... });:
 - Этот код добавляет обработчик события mousemove  к элементу tabNavList.
 - Событие mousemove срабатывает каждый раз, когда мышь перемещается над элементом.

* (drag) => { ... }:
 - Это стрелочная функция, которая будет вызвана каждый раз, когда произойдет событие mousemove.
 - Параметр drag представляет собой объект события, который содержит информацию о событии, такую как координаты мыши и движение мыши.

* if (!activeDragg) return;
 - Эта строка проверяет, активен ли флаг activeDragg.
 - Если activeDragg имеет значение false (или любое значение, приводимое к false), функция немедленно завершает выполнение и ничего не делает.
 - то условие используется для предотвращения выполнения кода прокрутки, если activeDragg не активен.

* tabNavList.scrollLeft -= drag.movementX;:
 - Если activeDragg активен, эта строка изменяет свойство scrollLeft  элемента tabNavList.
 - scrollLeft определяет горизонтальную прокрутку элемента. Уменьшая его значение на drag.movementX, мы прокручиваем элемент влево на расстояние, равное горизонтальному перемещению мыши.
 - drag.movementX — это свойство объекта события, которое указывает на количество пикселей, на которое мышь переместилась по горизонтали с момента последнего события mousemove.


2. Добавление обработчика события mousedown

tabNavList.addEventListener("mousedown", () => {
  activeDragg = true;
});

* tabNavList.addEventListener("mousedown", () => { ... });:
 - Этот код добавляет обработчик события mousedown  к элементу tabNavList.
 - Событие mousedown срабатывает каждый раз, когда кнопка мыши нажата над элементом.

* () => { ... }:
 - Это стрелочная функция, которая будет вызвана каждый раз, когда произойдет событие mousedown.

* activeDragg = true;
 - Эта строка устанавливает флаг activeDragg в true.
 - Это означает, что после нажатия кнопки мыши флаг activeDragg будет активен, и последующие события mousemove будут обрабатываться для прокрутки элемента.

 Общий смысл кода: 
 Этот код позволяет реализовать функциональность перетаскивания для горизонтально 
 прокручиваемого элемента tabNavList. 
 Когда пользователь нажимает кнопку мыши (mousedown) над элементом, флаг activeDragg устанавливается в true. 
 Затем, когда пользователь перемещает мышь (mousemove), элемент прокручивается влево или вправо 
 в зависимости от движения мыши, но только если флаг activeDragg активен. 
 Это позволяет пользователю перетаскивать элемент для его прокрутки.
